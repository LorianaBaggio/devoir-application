Découverte de WordPress : Chapitre 7, Gestion de la page d'accueil

Par défaut WordPress affiche une liste d'actualités lorsque l'on se rend sur la page d'accueil. Ceci peut convenir dans le cadre d'un blog mais pas nécessairement dans le cadre d'un site où on souhaitera afficher une page spécifique en tant que page d'accueil.
