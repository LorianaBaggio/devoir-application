Découverte du CSS : Chapitre 8, Les éléments flottants : float

Par défaut la pluspart des éléments (ceux en display: block) vont s'empiler les uns sous les autres. C'est un comportement que l'on voudra parfois contourner pour juxtaposer des éléments horizontalement. On pourra alors avoir recours aux éléments flottants qui permettent de sortir un élément du flux normal pour le placer à droite ou à gauche de son conteneur.
