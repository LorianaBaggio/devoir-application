Le préprocesseur SASS : Chapitre 1, Présentation de SASS

Je vous propose dans cette formation de découvrir ensemble le préprocesseur SASS. SASS est un préprocesseur qui vous propose de travailler avec une syntaxe CSS améliorée qui offre des fonctionnalités supplémentaires comme par exemple la possibilité d'imbriquer les règles, de concaténer plusieurs fichiers ensemble ou de créer des fonctions afin de limiter la répétition.
