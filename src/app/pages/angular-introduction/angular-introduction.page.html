Formation AngularJS : Introduction

AngularJS est un framework javascript qui permet de créer des application web dynamiques. Ce type d'applications (souvent appellées SPA pour Single Page Application) sont de plus en plus présentes avec des périphériques connectés de plus en plus variés.
Malheureusement, créer une application tenant sur une seule page n'est pas chose simple. En effet, cela requiert l'utilisation de beaucoup de javascript et il est très difficile de bien s'organiser et d'obtenir une application maintenable et modulable. Et c'est ici qu'AngularJS intervient.
